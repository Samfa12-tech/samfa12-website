Pocket DJ Help

Quick controls for performing a Pocket Chordsmith song.
ImportPaste a Pocket Chordsmith song export, then press Import. Use Load Demo to test the deck immediately.
Play sectionsTap a section pad to queue it. The current pad glows cyan. The queued pad pulses pink.
Set the Drop sectionLong-press a section pad. It will show a DROP badge and the button changes to Drop → section.
Build and DropBuild strips the mix back and adds tension. Drop restores the mix and lands on the drop section.
Launch modeNext Bar is DJ-style. End of Section finishes the whole section first. Instant jumps now.
SequencePlay Sequence follows the song order. Repeat loops that order.
Hold SectionUse this to stay on the current section during a sequence. Release it to let the next queued or sequence section play.
Stem MixerMute or lower drums, bass, chords, melody and guitar for live breakdowns and drops.
Edit this songCopies the original Pocket Chordsmith song and opens Chordsmith. DJ mutes, loops and FX stay in Pocket DJ.
Pocket Audio CoreThis build uses Pocket Audio Core for shared project parsing, timeline diagnostics and mirrored deck controls when the local core module is available. The legacy DJ audio engine remains active as the playback fallback.
Pocket DJ keeps the technical import details in the background so the deck stays focused on performance.
